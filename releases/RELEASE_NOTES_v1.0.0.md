# Vinilos v1.0.0

Primera versión liberada de la aplicación Vinilos.

## Funcionalidades

- Pantalla Home con accesos a Albums Catalogue y Artists List.
- Catálogo de álbumes con vista de grilla en 2 columnas.
- Carga de datos desde backend BackVynils vía API REST.
- Navegación entre Home y Albums Catalogue.
- Manejo de estados de carga y error.

## Pruebas

- 3 pruebas E2E automatizadas con Espresso (CP-01, CP-02, CP-03).
- 7 casos de prueba manuales ejecutados (CP-04 a CP-10).

## Requisitos

- Android 16.0 (API 36) o superior.
- Backend BackVynils corriendo en localhost:3000 (configurable).
