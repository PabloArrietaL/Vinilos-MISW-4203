# Vinilos v2.0.0

Segunda versión liberada de la aplicación Vinilos.

## Nuevas funcionalidades

- Listado de artistas
- Detalle de artista

## Mejoras

- Mejora en la visualización de las imágenes.
- Mejoras en el código

## Pruebas

- 5 Pruebas E2E automatizadas con Espresso (CP-01 a CP-05).
- 7 casos de prueba manuales ejecutados (CP-06 a CP-12).

## Requisitos

- Android 16.0 (API 36) o superior.
- Backend BackVynils corriendo en localhost:3000 (configurable).
